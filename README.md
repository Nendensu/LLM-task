### Контекст

### Подзадача 1

Используя **только NumPy** (никаких PyTorch, TensorFlow, sklearn):

1. Реализуйте функцию `forward(X, weights)`, возвращающую предсказания сети.
2. Убедитесь, что ваш forward pass воспроизводит эталонные предсказания из файла `reference_predictions.npy` (предоставляется) с точностью до $10^{-5}$.
3. Реализуйте функцию `compute_gradients(X, y, weights)`, возвращающую градиенты по всем параметрам.
4. Верифицируйте градиенты методом конечных разностей (finite difference check):

$$\frac{\partial L}{\partial w_i} \approx \frac{L(w_i + \varepsilon) - L(w_i - \varepsilon)}{2\varepsilon}, \quad \varepsilon = 10^{-5}$$

Разница между аналитическим и численным градиентом должна быть < $10^{-4}$. Приведите численные результаты проверки в отчёте.

### Подзадача 2*

1. Для **каждой** точки тестовой выборки, которая классифицируется правильно, найдите **минимальное по норме $\ell_2$ возмущение** $\delta$, при котором модель меняет решение:

$$\min_{\delta} \|\delta\|_2 \quad \text{s.t.} \quad \hat{y}(x + \delta) \neq \hat{y}(x)$$

Используйте итеративный градиентный метод (например, PGD или простой gradient ascent по входу).

2. Постройте scatter-plot исходных точек, наложив на него **границу принятия решений** сети и противоречивые примеры (adversarial examples). Где концентрируются наиболее уязвимые точки?

3. Ответьте письменно: почему нейронные сети уязвимы к таким атакам? Как связана эта уязвимость с геометрией пространства признаков?
